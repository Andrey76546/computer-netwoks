1️⃣ Шаг 1. Создание дерева каталогов
```cmd
cd C:\Temp
md Lab1
cd Lab1
md info A1 A2 A3
2️⃣ Шаг 2. Навигация и подкаталоги
md info\Personal info\University info\Hobby
md A1\B1 A1\B2 A3\B3
3️⃣ Шаг 3. Файлы в папке Personal
cd info\Personal
copy con Name.txt
Андр ИЛ
^Z
copy con Date.txt
23.05.2026
^Z
copy con School.txt
MFUA
^Z
4️⃣ Шаг 4. Файлы в папке University
cd ..\University
copy con Name.txt
Андр ИЛ
^Z
copy con Mark.txt
Отлично
^Z
5️⃣ Шаг 5. Файл в папке Hobby
cd ..\Hobby
copy con hobby.txt
Программирование
^Z
6️⃣ Шаг 6. Копирование и переименование
copy hobby.txt ..\..\A2
cd ..\..\A2
ren hobby.txt Lab_1.txt
7️⃣ Шаг 7. Создание копии и её удаление
copy Lab_1.txt copy_Lab_1.txt
del copy_Lab_1.txt
8️⃣ Шаг 8. Очистка экрана
cls
9️⃣ Шаг 9. Вывод содержимого Personal
cd ..\info\Personal
type Name.txt
type Date.txt
type School.txt
🔟 Шаг 10. Сортировка файлов
dir /on
1️⃣1️⃣ Шаг 11. Объединение файлов в all.txt
copy *.txt all.txt
type all.txt
1️⃣2️⃣ Шаг 12. Добавление года рождения

type all.txt
1️⃣3️⃣ Шаг 13. Копирование all.txt в A1
copy all.txt ..\..\A1
1️⃣4️⃣ Шаг 14. Удаление каталогов (с буквой А или цифрой 2)
cd ..\..

📌 Справочник примененных команд

| Команда | Описание действия |
| :--- | :--- |
| md | Создание новой директории |
| cd | Переход между папками |
| rd /s /q | Удаление каталога вместе с содержимым |
| copy con | Запись текста с клавиатуры в новый файл |
| copy | Копирование файлов и их склейка |
| ren | Изменение имени файла |
| del | Удаление файла |
| type | Вывод текста из файла на экран |
| echo >> | Добавление строки в конец файла |
| dir /on | Вывод файлов в алфавитном порядке |
| cls | Очистка экрана консоли |
| prompt | Настройка строки приглашения |
